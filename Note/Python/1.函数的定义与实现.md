#### 2.关键字传参(形参传参)

```python
#函数定义
def health_check(name,age,height,weight,hr,hbp,lbp,glu):
    print("您的健康状况你良好")

#函数调用（关键字可以不按原来的顺序）
health_check(name = "张三",age = 20,height = 180,weight = 130,hr = 70,hbp = 120,lbp = 80,glu = 4.3)

```


#### 3.混合形式传参
参数较多时，调用函数如果不传入关键字，比较难以阅读。因此在定义函数时候，可以强制要求调用函数时必须使用关键字传参

`*` 代表之后所有的参数传参时必须使用关键字传参

```python
def health_check(name,age,*,height,weight,hr,hbp,lbp,glu):

#函数调用（关键字可以不按原来的顺序）
health_check("张三",20,height = 180,weight = 130,hr = 70,hbp = 120,lbp = 80,glu = 4.3)
```

### 函数的使用技巧-2

#### 1.序列传参(列表传参)

调用函数时，在传入的列表参数前加`*`号，列表中元素个数需要按照从左往右顺序与函数形参对应。

```python
def  calc(a,b,c):
    return (a+b)*C

l = [1,5,10]
print(calc(*l))

```

### 2.字典传参

调用函数时，在传入的列表参数前加`**`号，列表中元素个数需要根据键（字典中冒号左边的称为**键**，冒号右边称为**值**）与函数形参对应,顺序可以不对应。

```python
def health_check(name,age,*,height,weight,hr,hbp,lbp,glu):


param = {'name':'张三', 'age':20, 'height':180, 'weight' : 130, 'hr' : 70, 'hbp' : 120, 'lbp':80, 'glu':4.3}
health_check(**param )
```
